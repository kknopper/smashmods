this mod goes over any costume, or you can add it as
an additional costume!

if you are adding this costume as an additional slot, you
MUST name the cXX folder in "kassar" the same number as
the additional slot or else the mod will always freeze!

this mod is also TEXIDfixed, meaning it will work properly with other
Peachs on screen
